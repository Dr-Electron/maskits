# Maskits
Configure your character and mint it as an NFT.

## Development
### Prerequisites
- [Rust](https://www.rust-lang.org/)
- [Dioxus CLI](https://github.com/DioxusLabs/cli)

### Serve
You can start a local development server, by doing:

```
dioxus serve
```
